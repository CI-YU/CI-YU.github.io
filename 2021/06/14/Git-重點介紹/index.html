版本管控的種類

  • 集中式版本管控
    • 特定的伺服器才會有完整的版本庫及儲存庫
    • 所有人的版本全部集中放在某台電腦管理
    • 常見的版控系統
      • TFS
  • 分散式版本管控
    • 團隊中每個人都擁有完整的版本庫或儲存庫
    • 常見的版控系統
      • Git
      • Mercurial
