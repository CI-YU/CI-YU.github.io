    作業系統：macOS Big Sur 11.2.3

robo 3T 無法安裝

執行下列指令

sudo xattr -r -d com.apple.quarantine /Applications/Robo\ 3T.app

使用 Mongodb 的奇怪的警告訊息

會要求將下方的屬性開啟，如想了解原因，點擊下方連結

useUnifiedTopology: true

警告細節

_id 怎麼來的

總長度為 12-byte 為以下部分組成

  • 4-byte，時間紀錄以 Unix 時間為準
  • 5-byte，隨機數
  • 3-byte，為遞增計數器，初始化為隨機數

可以透過 ObjectID 取得

const {ObjectID}=require(“mongodb”)
const id = ObjectID()

findOne and Find

findOne回傳值為callback

find回傳值為Cursor，好處是可以隨需求回傳特定資料就好，不需要全部回傳
