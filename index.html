nuxt 目錄架構

  • assets：專門放需要被 webpack 打包的資源，例如你的 sass 、 images
  • components：頁面的最小單位，就是一般我們做 SPA 會放的元件們
  • layouts：頁面版型，例如 navbar、 footer
  • middleware：進入任何一個網頁前需要做的驗證函式
  • pages：每個頁面放的地方，一個 .vue 就是一個網頁
  • plugins：需要注入 Vue instance 的插件，譬如你選擇的 UI framework
  • static：靜態資源，不需要被 webpack 打包的資源
  • store： Vuex
    參考資料：https://medium.com/@jackercleaninglab/ssr-nuxt-js-超入門-84a0823b45ed
